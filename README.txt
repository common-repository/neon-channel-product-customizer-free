<h4>Overview</h4>

The ultimate app for selling custom signs with WooCommerce.
Supports a wide range of products such as neon signs, LED signs, acrylic signs, metal signs, 2D and 3D channel letter signs.

NCPC allows you to extend your business of personalization of neon signs by offering you a configurator to allow your customers to customize online prior to order their custom neon sign. Easy and quick to set up.

<h4>Features</h4>
<ul>
<li>Custom TTF fonts support</li>
<li>Custom colors palette for limited colors usage</li>
<li>Definition of customization sizes and number of characters per text</li>

<div class="block-content"><h4>Minimum Requirements</h4>

<p><strong>This section describes how to install the plugin and get it working</strong></p>

<h4>Automatic installation (easiest way)</h4>

<p>To do an automatic install of Neon Channel Product Customizer Free, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.</p>

<p>In the search field type "Neon Channel Product Customizer Free" and click Search Plugins. Once you have found it you can install it by simply clicking "Install Now".</p>

<h4>Manual installation</h4>

<p><strong>Uploading in WordPress Dashboard</strong></p>

<ol>
<li>Download <code>neon-channel-product-customizer-free.zip</code></li>
<li>Navigate to the 'Add New' in the plugins dashboard</li>
<li>Navigate to the 'Upload' area</li>
<li>Select <code>neon-channel-product-customizer-free.zip</code> from your computer</li>
<li>Click 'Install Now'</li>
<li>Activate the plugin in the Plugin dashboard</li>
</ol>

<p><strong>Using FTP</strong></p>

<ol>
<li>Download <code>neon-channel-product-customizer-free.zip</code></li>
<li>Extract the <code>neon-channel-product-customizer-free</code> directory to your computer</li>
<li>Upload the <code>neon-channel-product-customizer-free</code> directory to the <code>/wp-content/plugins/</code> directory</li>
<li>Activate the plugin in the Plugin dashboard</li>
</ol>

<p>The WordPress codex contains <a href="http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation" rel="nofollow">instructions on how to install a WordPress plugin</a>.</p>

<h4>Updating</h4>
